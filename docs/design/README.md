# Design Journal

Design explorations and research for gert features. Each topic folder contains a `design.md` with problem statement, proposed UX, technical notes, and collected examples.

## Status Legend

| Status | Meaning |
|--------|---------|
| collecting | Gathering screenshots, examples, API info |
| designing | Shaping the solution, open questions being resolved |
| ready | Design complete, ready to implement |
| implementing | Work in progress |
| done | Shipped |
